# "일당백, 학교 급식과 싸우는 노동자들"
기사 보러가기: [Link](https://euten11.wixsite.com/my-site-3)

우리는 **뜨거운 가스실**과도 같은 고된 노동 환경 속에서 일하는 '급식 조리사'분들을 심층 취재하고, 그 힘듦을 데이터로 표현하였습니다.
그들은 학생들의 점심식사를 책임지기 위해 1인 평균 **109.5명**의 식수를 책임지며, **40도**에 육박하는 조리실을 버텨냅니다.

더 큰 문제는 바로 **조리흄**입니다. 급식 식단의 **약 74.9%**가 조리흄이 다량 발생하는 메뉴로 이루어져 있으며, 이는 **심각한 폐질환**을 야기합니다.
급식 조리사의 40%가 폐질환 소견을 받은바 있으며, **폐암 확진률**은 약 0.27%로, 35-64세 여성 인구 10만 명당 폐암 발생률 0.028%의 거의 **10배**에 달합니다.
그럼에도 불구하고, 환자가 가장 많았던 서울 학교들의 2025년 환기시설 정비 이행률은 11%에 불과하였습니다.

데이터짱이 급식 만족도 조사와 급식 조리사 1인당 식수인원의 상관관계를 분석한 결과 **급식 조리사의 근무환경이 만족스러운 급식을 제공한다는 사실**을 알 수 있었습니다.
학교 급식이 붕괴하고 있습니다. 자랑스러운 K-급식은 급식 조리사분들의 희생아래 간신히 유지되고 있습니다.

**급식실 개선은 아이들을 위한 것입니다.** 근본적인 변화가 필요합니다.
